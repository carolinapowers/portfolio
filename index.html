I want to create experiences that satisfy people's needs

In October of 2014, I decided to make a change. I resigned from my job as a Physical Education teacher and invested my time and energy in learning web development in an intensive coding bootcamp. Why? Because I realized that people were passionate about web applications just like I was, and I wanted to be part of the process of making peoples' wishes come true.

Web development is not only about creating and maintaining websites. Web development is about researching the latest technologies, interacting with your fellow developers and designers, thinking about user interactions, and delivering the best solution to your clients. I’m looking for a position as a front-end developer where I can be part of that process.

Education

I learned web development through a 3 month intensive bootcamp at The Iron Yard. From the basics of HTML and CSS to the more complex AngularJS, Firebase, and Git, I learned the latest web development tools and technologies while following an agile & scrum development process.

My Projects

whatsPup

This was my final project at The Iron Yard. I worked with a partner to create a web application for pet sitters to provide real-time updates to their clients. To complete the project I used AngularJS, AngularFire, Firebase, Bootstrap, Sass, Gulp, Browser-sync, Yeoman, npm, Ajax, Madrill API, Cloudinary API, Vanilla JavaScript, and Gitflow.

Check out the live application at whatspup.firebaseapp.com and check the code on github.

Do You Even Feel Bro?

I worked with a classmate to create a music player that plays playlists based on your mood. We used SoundCloud’s API, jQuery, Ajax, HTML5, CSS3, Github, and Git to complete the project.

Check out the live application on my gh-pages at http://carolinapowers.github.io/TIY-Mashup/ and check the code on github.

Surf & Paddle

This was my first project at The Iron Yard. A responsive web site based around recreating a graphic designers’ template using HTML5, CSS3, Pixel Perfect, Skitch, Github and Git.

Check it out on my gh-pages at http://carolinapowers.github.io/TIY-Assignments/surf+paddle/index.html and check the code on github.
